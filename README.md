##  Descrição do Projeto
O projeto **NOUS** é uma aplicação voltada para **monitoramento de saúde** em tempo real, utilizando conceitos de **Internet das Coisas (IoT)**, **IoB (Internet of Behavior)** e **Inteligência Artificial**.  

Nesta Sprint, foi desenvolvido um **protótipo funcional** integrando:
- Um **fluxo IoT criado no Node-RED**, responsável por simular sensores de temperatura corporal e batimentos cardíacos;  
- Um **dashboard interativo no Oracle APEX**, exibindo as informações coletadas de forma visual e analítica.

---

##  Evolução da Sprint 2
Na Sprint 2, o grupo avançou na implementação prática do projeto:  
- Criação do **fluxo IoT** no **Node-RED** (`iot_flow.json`) para gerar dados simulados;  
- Integração dos dados simulados ao **Oracle Cloud e APEX**;  
- Desenvolvimento de **gráficos dinâmicos** para visualização das métricas de cada integrante do grupo (temperatura e batimentos cardíacos);  
- Estruturação do **protótipo inicial completo**, consolidando o aprendizado sobre arquiteturas disruptivas.  

---

##  Tecnologias Utilizadas
- **Oracle APEX** – Criação da interface e dashboards  
- **Oracle Cloud Infrastructure (OCI)** – Hospedagem e banco de dados  
- **Node-RED** – Simulação de sensores IoT  
- **JavaScript / JSON** – Formatação e transmissão de dados  
- **MQTT (planejado para próxima sprint)** – Comunicação em tempo real  

---

##  Resultados
O sistema exibe as informações de **temperatura (°C)** e **batimentos cardíacos (BPM)** de cada integrante em um **gráfico comparativo no APEX**, permitindo monitorar a variação dos dados simulados.
